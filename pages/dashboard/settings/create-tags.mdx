## Steps

You can organise your buildings using tags. Tags can be reused on multiple buildings, and are useful for grouping multiple buildings in relation to responsible person ("Rolf") or location ("Bergen"). You can name tags freely.

<Steps>
    <Step title="Create tag">
        Select "Create tag" to begin.
        <Frame type="ghost">
            <img src="/images/dashboard/settings/create-tags/step-1.png" />
        </Frame>
    </Step>
    <Step title="Manage tags">
        Select one or more tags from the list, or add a new one. Click outside the menu to confirm the changes.
        <Frame type="ghost">
            <img src="/images/dashboard/settings/create-tags/step-2.png" />
        </Frame>
    </Step>
</Steps>

Selected tags are shown in the list of buildings.

</Frame>
