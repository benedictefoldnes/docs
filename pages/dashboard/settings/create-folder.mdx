## Steps

<Steps>
    <Step title="Create folder">
        Go to a floor in your building and select "Create folder".
        <Frame type="ghost">
            <img src="/images/dashboard/settings/create-folder/step-1.png" />
        </Frame>
    </Step>
    <Step title="Give folder a name">
        Write the desired name for the folder and click "Create".
        <Frame type="ghost">
            <img src="/images/dashboard/settings/create-folder/step-2.png" />
        </Frame>
    </Step>
    <Step title="Move draft to folder">
        To move a draft, you can drag and drop the draft over the new folder.
        <Tip>
            You can also click on the 'dots' and select "Move to folder". Select the correct folder in the overview and click "Confirm" to move the draft.
        </Tip>
        <Frame type="ghost">
            <img src="/images/dashboard/settings/create-folder/step-3.png" />
        </Frame>
    </Step>
</Steps>

### Change folder name

You can change the name of the folder by clicking on the 'dots' and selecting "Change name".

<Frame type="ghost">
    <img src="/images/dashboard/settings/create-folder/change-name.png" />
</Frame>

### Remove draft from folder

To move the draft back to the main page, you can navigate to the folder and click "Remove from folder" in the menu.

<Frame type="ghost">
    <img src="/images/dashboard/settings/create-folder/remove-from-folder.png" />
</Frame>
