When you activate a floor plan, the floor plan becomes editable. You can make changes to areas and furniture, as well as export statistics and the floor plan in multiple formats.

<Frame type="ghost" caption="Example of an activated floor plan in the editing tool.">
    <img src="/images/dashboard/digitalisation/digitalisation.png" />
</Frame>

## Fremgangsm√•te

<Steps>
    <Step title="Activate floor">
        To create an editable version, you need to start an activation. Click "Activate floor".
        <Frame type="ghost">
            <img src="/images/dashboard/digitalisation/step-1.png" />
        </Frame>
    </Step>
    <Step title="Select 'Digitalisation'">
        Select "Digitalise floor plan". Click "Next".
        <Frame type="ghost">
            <img src="/images/dashboard/digitalisation/step-2.png" />
        </Frame>
    </Step>
    <Step title="Upload file">
        Upload a PDF (or DWG) file of the floor plan, or select the correct file if you have already uploaded it.
        <Frame type="ghost">
            <img src="/images/dashboard/digitalisation/step-3.png" />
        </Frame>
    </Step>
    <Step title="Confirm digitalisation">
        Review the details and click "Confirm" to start digitalisation of the floor plan.
        <Frame type="ghost">
            <img src="/images/dashboard/digitalisation/step-4.png" />
        </Frame>
    </Step>
</Steps>

The digitalisation is in progress. We recommend you to turn on [Notifications](/pages/dashboard/settings/notifications), so you get a notification when the floor plan is available.
