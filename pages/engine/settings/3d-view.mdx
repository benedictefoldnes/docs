3D-view is an additional feature that is used to create a better room experience. We recommend to furnish the area for the best possible experience.

## Steps

<Steps>
  <Step title="Open 3D view">
    Go to the toolbar and click "3D".
    <Frame type="glass">
        <img src="/images/engine/settings/3d-view/step-1.png" />
    </Frame>
  </Step>
  <Step title="Juster visningsinnstillinger">
    Select view type and adjust details for the view in the left sidepanel.
    <Frame type="glass">
        <img src="/images/engine/settings/3d-view/step-2.png" />
    </Frame>
  </Step>
</Steps>

## Properties for 3D view

Here is an overview of the different settings for 3D view.

    | Property             | Description                           |
    | -------------------- | ------------------------------------- |
    | First person view   | Place camera in person height, navigate with "W" (forward), "S" (backward), "A" (left) and "D" (right). |
    | Orthographic camera   | Flat view without depth distortion. Ideal for architectural presentation. |
    | Half height on walls | Cut all walls in height 120 cm. |
    | Show doors and windows | Hide doors and windows. |
    | Show interior         | Hide furniture. |
    | Show shadows          | Turn shadows on and off. |
