## Steps

<Steps>
  <Step title="Select window tool">
    From the toolbar at the top, select the "Window" tool.
    <Tip>
      You can also use the keyboard shortcut "Q".
    </Tip>
    <Frame type="glass">
        <img src="/images/engine/tools/add-window/step-1.png" />
    </Frame>  
  </Step>
  <Step title="Place window">
    <Info>
      The window width will be adjusted from the center.
    </Info>
    Use the mouse to move the window to the correct position, and click to confirm placement.
    <Frame type="glass">
        <img src="/images/engine/tools/add-window/step-2.png" />
    </Frame>
  </Step>
  <Step title="Configure window">
    Use the left sidebar to set the width of the window, and change the height and distance from the floor if desired.
    <Frame type="glass">
        <img src="/images/engine/tools/add-window/step-3.png" />
    </Frame>
  </Step>
</Steps>
