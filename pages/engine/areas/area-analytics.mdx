## General area overview

<Steps>
  <Step title="Open analytics panel">
    In the left sidebar, open the "Analytics" tab.
    <Frame type="glass">
        <img src="/images/engine/areas/area-analytics/step-1.png" />
    </Frame>
  </Step>
  <Step title="Read analytics">
    See overview over:
    - Number of fixed workstations
    - Area per employee
    - Number and total area of each room type
    - Total exclusive area (BTA)
    - Common area - Floor
    <Frame type="glass">
        <img src="/images/engine/areas/area-analytics/step-2.png" />
    </Frame>
  </Step>
  <Step title="Adjust common area surcharge">
    By writing in the common area surcharge(%), you will see the total lease area. 
    <Frame type="glass">
        <img src="/images/engine/areas/area-analytics/step-3.png" />
    </Frame>
  </Step>
</Steps>

## Area for room type

<Steps>
  <Step title="Select a room type">
    Select a room type from the pie chart to select the area in the drawing.
    <Frame type="glass">
        <img src="/images/engine/areas/area-analytics/step-4.png" />
    </Frame>
  </Step>
  <Step title="Read area">
    You can read NTA and BTA area in the left sidebar.
    <Frame type="glass">
        <img src="/images/engine/areas/area-analytics/step-5.png" />
    </Frame>
  </Step>
</Steps>

## Exclusive area / Common area

Mark room types from the pie chart and set the area type to "Common area" to separate them out in the statistics.

<Frame type="glass">
    <img src="/images/engine/areas/area-analytics/step-6.png" />
</Frame>

## Download statistics

You can download a list containing room types, room names and room numbers. 

<Steps>
  <Step title="Download excel file">
    Click the "Download file" button to download an Excel file to your downloads folder.
    <Frame type="glass">
      <img src="/images/engine/areas/area-analytics/step-7.png" />
    </Frame>
  </Step>
  <Step title="Open file">
    Open the file to see the overview.
    <Frame type="glass">
      <img src="/images/engine/areas/area-analytics/step-8.png" />
    </Frame>
  </Step>
</Steps>
